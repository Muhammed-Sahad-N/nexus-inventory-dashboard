# Nexus Global Logistics - Automated Inventory Dashboard

An interactive supply chain dashboard designed to bridge the gap between inventory monitoring and business workflow execution. This project highlights practical data modeling, dynamic UI controls, and workflow automation within an enterprise logistics framework.

## Key Features & Functionality
- **Interactive Metrics Tracking:** Utilizes Power Query to clean and consolidate logistics datasets, tracking revenue performance and global stockout metrics.
- **Dynamic Interface Navigation:** Features multi-layered visual slicers for cross-filtering data grids alongside a custom macro to instantly reset active user views.
- **Automated Alert System:** Implements a clean VBA script that automatically loops through records to identify safety stock deficits, compiling the results into a text summary and launching a structured email draft for internal procurement teams.
